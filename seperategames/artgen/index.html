gen type: 
q - delete color
w - add color

g - generate

o/l - width up/down
p/; - height up/down

< - size down
> - size up 

1-9 select color
rmb - select color from picture
lmb - draw on picture

Modes:
    random | randomly sets pixels
    random + probability | set chance for each pixel 2.5 = 250%, 0.3 = 30%...
    groups | 0 = no group, use any non zero number to set groups of pixels
